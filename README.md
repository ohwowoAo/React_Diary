### 프로젝트 소개

총 4개의 페이지 (일기 목록, 새 일기작성, 일기 수정, 일기 내용보기)로 구성되어 있습니다.

- 총 4페이지 (일기 목록, 새 일기작성, 일기 수정, 일기 내용보기) 구현
